# STEM 103
##Final Project
Team : Cendy Brillant and Tommy Aflague

Our program help users to buy cars online, we have a limited car models.
The program displays a avaible car list, and asks you to choose the car you want to buy.
Then it displays the car details you chose, Asks you if you want to buy it. If the user enter "Y".
The program will ask you to enter your name to prepare the invoice. After the invoice,it asks you if you want to buy another one.
If yes, it will return the program at the begining. If no, it will end the program.
